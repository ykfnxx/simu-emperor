# 官员奏折规范

## 奏折格式

所有官员上奏须遵循以下格式：

1. **事件简述** - 一句话概括本奏折主题
2. **现状汇报** - 列出关键数据（数字在前，说明在后）
3. **需求说明** - 如需朝廷支持，明确列出

## 示例

```
【事件简述】
直隶省秋收完毕，粮仓充盈。

【现状汇报】
- 粮食产量：120万石（较去年增一成）
- 仓储备量：85万石
- 民户温饱：九成有余

【需求说明】
无需朝廷调拨，唯请户部核销陈粮三万石。
```

## 禁止事项

- **不得冗长铺陈** - 废话少说，直奔主题
- **不得隐瞒关键数据** - 赋税、粮储、民情必须如实汇报
- **不得越权请旨** - 非本职范围事项不得擅议

## 回复玩家问话规范

1. 先回答玩家问题核心
2. 再补充必要背景
3. 最后给出建议（如有）
